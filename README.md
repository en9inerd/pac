I mean I'll try to handcraft :melting_face: but using a plan generated by nerfed Claude Opus model lol and... using [Beej's Guide to Network Programming](https://beej.us/guide/bgnet/html/) and man pages :upside_down_face:

# Stack
- pure C... C23 standard
- Zig's build system
- Go web server (WebSocket <-> raw TCP bridge) + JS browser client
- C CLI client - raw TCP, line-based
- libsodium for cryptographic stuff (it will be too much to implement my own)
- MessagePack - binary serialization, multi-language safe
- mpack.h - vendored single-header MessagePack for C
- noise tunnel - optional DPI bypass (obfuscates traffic, resistant to active probing)
