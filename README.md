# Stack
- pure C... C23 standard
- Zig's build system
- Go + JS - web client, bridges browser WebSocket ↔ raw TCP
- libsodium for cryptographic stuff (it will be too much to implement my own)
- MessagePack - binary serialization, multi-language safe
- mpack.h - vendored single-header MessagePack for C
